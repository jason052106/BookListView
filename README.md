# BookListView - 圖書管理程式

本專案是一個基於 C# Windows Forms 開發的視覺化圖書管理與借閱系統。核心功能著重於 **ListView（清單檢視控制項）** 的深度應用與進階操作，並結合了多種檢視模式切換、動態資料匯入優化，以及完善的借還書互動邏輯。

---

##  核心功能與技術特點

### 1. 多元檢視模式切換 (`View` 屬性)
支援透過下拉式選單 (`ComboBox`) 動態變更書籍清單的呈現方式，完整實作五種內建檢視模式：
*  **大圖示 (LargeIcon)**：結合 `LargeImageList` 呈現精美書籍封面。
*  **詳細資料 (Details)**：以多欄位（書名、作者、類別）形式呈現，支援指定欄位寬度（如書名欄位寬度 100，其餘欄位寬度 60）。
*  **小圖示 (SmallIcon)**：結合 `SmallImageList` 節省視窗空間。
*  **清單 (List)**：採單行式項目橫向或縱向排列。
*  **大圖示加詳細資料 (Tile)**：兼具大圖示視覺與基礎欄位資訊。

### 2. 高效能畫面重繪優化
批次加入大量書籍資料時，系統自動啟用 `BeginUpdate()` 暫停畫面重繪，待所有 `ListViewItem` 與 `SubItems` 新增完畢後，再調用 `EndUpdate()` 進行一次性重繪。
* **技術效益**：徹底消除逐筆插入資料時產生的螢幕閃爍現象（Flicker），優化使用者體驗。

### 3. 直覺式雙擊借書機制 (`ItemActivate` 事件)
使用者雙擊 `ListView` 中的任何書籍，即會觸發項目啟動事件：
* 連動 `SelectedIndices` 陣列精確定位目前選取書籍。
* 內建重複檢查機制，若書籍已存在於「借書清單」，則自動忽略以防止重複借閱。
* 使用 `MessageBox.Show` 彈出互動視窗，進行二次確認（是/否），確保操作精準。

### 4. 嚴謹防呆之歸還功能 (`DoubleClick` 事件)
全新擴充的還書功能，使用者可直接在 `ListBox` (借書清單) 中雙擊欲歸還的書籍：
* **嚴格防呆機制**：程式優先驗證 `SelectedIndex != -1`。確保清單有實際選取項時才執行邏輯，徹底杜絕空指標或索引超出範圍 (`IndexOutOfRangeException`) 導致的程式崩潰。
* 透過 `RemoveAt(SelectedIndex)` 實現即時、動態的記憶體清單資料移除。

## 🛠️ 控制項與變數規劃

### 畫面主要控制項
* `frmBooks`：主視窗表單，設定為 `TwoClick` 啟動模式。
* `lvwBooks`：書籍顯示清單 (`ListView`)，綁定雙影像控制項。
* `lstBorrow`：借書清單 (`ListBox`)，用於存放目前已借閱的書名。
* `cmbView`：檢視方式下拉選單 (`ComboBox`)。
* `imgL` / `imgS`：影像清單控制項 (`ImageList`)，定義圖示尺寸。
